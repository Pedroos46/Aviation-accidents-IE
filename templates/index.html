Information Extraction: Aviation Accidents.

Authors: Roger Pedrós Villorbina. Wenjie Jin. Rodrigo Caero.

As part of our Intelligent Applications for the Web subject in UPF we have been learning about information extraction. We have seen different ways of extracting and understanding information, up to advanced solutions based on BERT.

As part of this work we've been working on a BERT model which we fine-tune to extract certain information. In our case, we work with an aviation accidents data set from Concisus corpus a bilingual (English/Spanish) corpus for Information Extraction. So our fine-tuned BERT model should recognise tags like: DateOfAccident, TypeOfAircraft, Place, Victims, etc, as specified in the full report

Finally for this application, we have built a small web application in order to test it more comfortably to test it. Ideally it should be started together with the Flask server which contains the model.
